# YS_MENGDECHENG-DEMO
一款个人开发的基于原神的二创游戏DEMO，地图仅限于蒙德城且尚未全面完善，部分贴图未上，UI界面未完善,人物基础移动视角控制功能已完成，按数字1-3可切换角色和官方的游玩机制一样，有需要可自行下载进行完善练手,渲染管线为HDRP，版本最好是2022LTS及以后
